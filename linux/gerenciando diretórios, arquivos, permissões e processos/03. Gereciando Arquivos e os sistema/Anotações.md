03. Removendo diretórios e redirecionando saídas para arquivos 

```bash
    rm # Remove pasta 

    rm -r # Remove arquivos de maneira recursiva

    rm -ri # Remove arquivos de maneira recursiva pergutando quais arquivos ele quer deletar

    rmdir # Remover pasta vazias

    ls >> lista_projeto.txt # redireciona a saida para um destino >> siginifica apeend > um replace 
```
