# 04. Movendo arquivos e diretórios

```bash
    mkdir # Cria um novo diretório 

    mv diretorio_origem diretorio_destino # Move arquivo entre pastas

    ls -al # Mostre arquivos ocultos e suas permissões
```
drwxr-xr-x 3 root root 4096 May  4 22:13 . -> d: diretorio, r: permissão de leitura, x: permissão de exercução, w: permissão de escrita, xr -> permissões de grupo
